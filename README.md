# Remote IoT Sensor

## Year 2  Emerging Technologies Project
### IoT remote sensor which sends data via GPRS to a SQL database<br>

A solo project where I explored working with an arduino to send data remotely.<br>

#### Project Goals
*	Record Environmental data
*	Send data wirelessly
*	Store data
*	Simple platform to view data. <br>
